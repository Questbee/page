Features

Why teams choose Questbee

Everything you need to collect, sync, and own your field data — without depending on third-party clouds.

🔒

Data Sovereignty

Runs entirely on your infrastructure. No external dependencies, no cloud accounts, no vendor lock-in. Full compliance with data localization regulations.

📱

True Offline First

Collect data anywhere — even with zero connectivity. Local SQLite storage on the device. Automatic conflict-resolved sync when connection is restored.

🎨

White-Label Ready

Custom branding in mobile apps and web dashboards. Deploy with your organization's logo, colors, and domain name. Invisible to end users.

🤖

AI & IoT Ready

Headless API for IoT sensor ingestion. MCP server support for AI agents to generate forms, inject test data, and analyze submissions.

⚡

Fast Deployment

Single docker-compose up command. Deploy in minutes on any Linux server, AWS, Azure, or private on-premise infrastructure.

📊

Rich Data Capture

GPS coordinates, photos, barcode scanning, digital signatures, audio recordings, and advanced conditional form logic — all offline.

Migration support is included in the Implementation service package. The Questbee schema is inspired by XLSForm (used by KoboToolbox/ODK), making form migration straightforward. Historical data migration from existing deployments is handled case-by-case.

White-labeling (Enterprise only) covers: custom logo and colors in the web dashboard, a fully branded mobile app with your organization's name published to app stores, a custom domain, and removal of all Questbee branding from user-facing interfaces.
